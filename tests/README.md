# Tests Folder

This folder contains manual test pages, sample scan files, and utility scripts used for validation and troubleshooting.

## Scope

- Browser test pages (`test-*.html`)
- Sample CSV/TXT inputs (`test_*.csv`, `test_*.txt`)
- Utility scripts used for local verification

## Notes

- These files are not part of the production runtime path.
- Keep test artifacts in this folder (not repo root).
- Use `index.html` for the main application and `tests/test-skills-pipeline.html` for skills pipeline validation.
